# CLO835 Group Project

This group project is a cloud application deployment assignment for CLO835. We built a Flask employee management web application with a MySQL backend, containerized both services with Docker, and deployed them on Amazon EKS using Kubernetes manifests. The project includes CI/CD with GitHub Actions and Amazon ECR, configuration management with ConfigMaps and Secrets, persistent storage using PVC, and optional autoscaling using HPA.
